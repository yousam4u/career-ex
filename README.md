# You쌤 퀴즈 보드

초등학생·중학생 수업에서 바로 사용할 수 있는 클릭형 퀴즈 랜딩페이지입니다.
문제 카드를 누르면 정답이 크게 표시되고, 관리자 모드에서 문제를 직접 수정하거나 주제 키워드로 문제 초안을 만들 수 있습니다.
저장한 퀴즈는 SQLite 데이터베이스(`data/quizzes.db`)에 별도로 관리됩니다.

## 실행 방법

Python만 있으면 별도 패키지 설치 없이 실행할 수 있습니다.

```powershell
cd "G:\내 드라이브\10_Agent_Team\Career-Ex"
python server.py
```

브라우저에서 아래 주소를 엽니다.

```text
http://127.0.0.1:8000
```

## 관리자 사용 방법

1. 오른쪽 위 톱니바퀴 버튼을 누릅니다.
2. 비밀번호 `youssam`을 입력합니다.
3. 주제 또는 짧은 키워드를 입력하고 `문제 자동 만들기`를 누릅니다.
4. 필요한 문장을 직접 다듬은 뒤 `저장`을 누릅니다.
5. 저장된 퀴즈는 `저장된 퀴즈 불러오기` 목록에서 다시 불러올 수 있습니다.

퀴즈 편집 형식은 한 줄에 하나씩 입력합니다.

```text
문제 내용 | 정답 내용
```

## 주요 기능

- 큰 글씨 중심의 수업용 퀴즈 화면
- 문제 클릭 또는 `정답 보기` 버튼으로 정답 표시
- 이전/다음 문제 이동
- 효과음 내장
- 관리자용 간단 로그인
- 주제·대상·문제 수 기반 문제 초안 생성
- SQLite 데이터베이스 저장
- 저장된 퀴즈 목록 불러오기

## 주의사항

현재 관리자 비밀번호는 정적 페이지 안에 들어 있는 간단 잠금입니다.
학교·기관에서 실제 계정 관리가 필요하면 서버 인증 방식으로 바꾸는 것이 안전합니다.

## 데이터베이스 위치

서버를 처음 실행하면 아래 파일이 자동으로 생성됩니다.

```text
G:\내 드라이브\10_Agent_Team\Career-Ex\data\quizzes.db
```

## Vercel 배포 참고

Vercel 배포판은 `api/index.py` 서버리스 함수로 기본 퀴즈를 제공합니다.
Vercel 서버리스 파일시스템은 SQLite 파일을 영구 저장소로 쓰기에 적합하지 않으므로,
관리자에서 저장한 내용은 브라우저 저장소 중심으로 동작합니다.
온라인에서 여러 관리자가 같은 DB를 공유하려면 Neon/Postgres 같은 외부 데이터베이스 연결이 필요합니다.
